# VibeCoding

## Overview

VibeCoding is a simple web-based bubble popper game designed to improve cursor accuracy. This project was developed as part of the Vibe Coding Hackathon, which originally planned for three difficulty modules: easy, medium, and hard. Due to time constraints during the hackathon, only the easy module was fully implemented.

The game challenges players to pop bubbles by clicking on them, helping gamers enhance their mouse precision and reaction time, which can be beneficial for shooting accuracy in games.

## Features

- **Bubble Popper Gameplay**: Click on bubbles to pop them and score points.
- **Accuracy Training**: Designed to improve cursor control and clicking precision.
- **Simple Interface**: Clean, minimalistic design using HTML, CSS, and JavaScript.
- **Score Tracking**: Keep track of your performance with a scoring system.

## Why I Built This

I participated in the Vibe Coding Hackathon, which included modules categorized as easy, medium, and hard. The hackathon's time limit was very short, so I focused on completing the easy module to deliver a functional project. The bubble popper game serves as a practical tool for gamers to train their cursor accuracy, potentially improving their performance in fast-paced games that require precise aiming and clicking.

## Technologies Used

- **HTML**: For the structure of the game interface.
- **CSS**: For styling and visual elements.
- **JavaScript**: For game logic, bubble generation, and interaction.

## How to Start

1. **Clone or Download the Repository**: Ensure you have the project files (`index.html`, `script.js`, `style.css`) in a local directory.

2. **Open the Game**:
   - Locate the `index.html` file in your file explorer.
   - Double-click the file or right-click and select "Open with" your preferred web browser (e.g., Chrome, Firefox, Edge).

3. **Play the Game**:
   - The game will load in your browser.
   - Bubbles will appear on the screen.
   - Click on the bubbles to pop them and increase your score.
   - Try to pop as many as possible to improve your accuracy!
